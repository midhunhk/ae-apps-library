About libAeApps

LibAeApps is an Android library project which provides reusable solutions to some common problems.
Some solutions require boiler plate code which can become tedious to manage.

Features


A modern Android app library that targets the latest Android APIs.

Contains common reusable code which can be used in any Android project.

Uses AndroidX libraries.

Developed by Ae App Labs.

Functionalities are separated into different modules.
